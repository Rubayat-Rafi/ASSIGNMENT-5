What is the Document Object Model (DOM)?

Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

How do you select an element from the DOM?

To select an element from the Document Object Model (DOM) in JavaScript, (getElementById) to select by ID,(getElementsByClassName) to select by class name, (getElementsByTagName) to select by tag name, or the more flexible (querySelector) and (querySelectorAll) which allow for CSS selector-based selection to target elements based on various attributes and combinations of selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element, allowing you to handle events triggered on any of its child elements by leveraging the "event bubbling" mechanism; this is particularly useful when dealing with large lists of dynamically added elements, as it significantly improves performance by reducing the number of event listeners needed and simplifies code maintenance by centralizing event handling logic in one place.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, you first need to select the desired element, then access its attributes using setAttribute to change values and its style properties directly through the style property to modify CSS styles.
